# CPD_Proyecto_Evolucion_Galactica
Como parte del curso de Computación Paralela y Distribuida se realizo el análisis del proyecto Evolución Galáctica, que parte del desarrollo por el grupo Astrofísica de la Universidad de Heidelberg, Alemania. Para ello, se contó con el código hibrido de NCuerpos del cual se partió el análisis.
